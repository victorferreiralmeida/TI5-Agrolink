# AGROLINK — app Flutter

Projeto Flutter do mobile. A documentação completa está em:

**[`../README.md`](../README.md)**

Comandos rápidos (na pasta `code/mobile`):

```bash
npm install
npm run dev
```

Ou manualmente:

```bash
flutter pub get
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:8080
```
