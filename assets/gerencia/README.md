# Artefatos relativos à Disciplina de Gerência de projetos

Este diretório mantém os artefatos relativos à gerência do projeto. 

Os principais documentos produzidos são:

* [termo_de_abertura_do_projeto.md](./termo_de_abertura_do_projeto.md)
	* Termo de Abertura do Projeto AGROLINK contendo definição inicial do projeto, equipe, objetivos e escopo.

* [eap.md](./eap.md)
	* Estrutura Analítica do Projeto (EAP) contendo a divisão das atividades, entregas e organização do desenvolvimento.

Demais artefatos poderão ser adicionados conforme a evolução do projeto.
